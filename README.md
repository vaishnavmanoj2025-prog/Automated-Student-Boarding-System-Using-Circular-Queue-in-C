# Automated-Student-Boarding-System-Using-Circular-Queue-in-C
Smart Campus Shuttle Queue Management System in C using a circular queue. It manages student boarding efficiently with FIFO logic, allowing enqueue, dequeue, and display operations while optimizing memory usage and ensuring fair, real-time shuttle allocation.
🚍 Smart Campus Shuttle Queue Management System
📌 Overview

A C-based application that simulates a campus shuttle system using a circular queue. It manages student boarding efficiently using the FIFO (First-In-First-Out) principle.

⚙️ Features
➕ Join shuttle queue (Enqueue)
🚌 Board students based on available seats (Dequeue)
📋 Display current queue
🔁 Circular queue for efficient memory usage
⚠️ Handles overflow and underflow
🧠 Concepts Used
Circular Queue
FIFO Scheduling
Structures in C
Modular Programming
🛠️ Tech Stack
Language: C
Compiler: GCC / Turbo C
▶️ How to Run
gcc shuttle.c -o shuttle
./shuttle
📸 Sample Menu
CAMPUS SHUTTLE SYSTEM
1.) Join Queue
2.) Board Bus
3.) Display Queue
4.) Exit
🎯 Objective

To demonstrate real-world use of queue data structures in managing transportation systems efficiently.
